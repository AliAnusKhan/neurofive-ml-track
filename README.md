# Neurofive ML Track

## Titanic Dataset - Exploratory Data Analysis (EDA)

This project was completed as part of the Neurofive Solutions ML Track.

### Tasks Completed
- Data inspection and exploration
- Missing value handling
- Outlier detection using boxplot
- Data visualization using Matplotlib and Seaborn
- Histogram
- Boxplot
- Bar Chart
- Correlation Heatmap
- Survival feature analysis

### Dataset
- Titanic Dataset

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

### Conclusion
The analysis showed that **Sex** has the strongest correlation with survival, followed by **Pclass**. Female passengers had a significantly higher survival rate than male passengers.
